Margins, Borders and Padding

Websites are often comprised of 3 main visual components. This includes Margins, Borders, and Padding.

These 3 components are used to position content on a page, a bit like wrapping up an item for shipping.

  • Padding: The area surrounding the content. Think of this like the bubble wrap you surround your item with.
  • Border: This outlines the content. Think of this like the cardboard box you might wrap your item in.
  • Margin: This defines your position within the page. Think of this like where your box sits, within the shipping container.
