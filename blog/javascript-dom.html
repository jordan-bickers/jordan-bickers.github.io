An introduction to Javascript and the DOM:

To begin with, let's have a look at what Javascript is, and how it relates to CSS and HTML:

HTML5 (The latest HML) is comprised of three componenets, HTML, CSS and Javascript. It may be helpful to think of these like a car. A cars structure comes from its chassis, which in this case would be HTML. This defines the basic shape and purpose of the car.

Then you have CSS which defines the style of the car. What shape is it? What color?

And finally there is Javascript, allowing you to interact with the car, use the pedals to make it move and the steering wheel to make it turn, etc.

Next we can look at Control Flow and Loops:

Control Flow is the order in which functions are executed. An example of Control Flow that you may experience everyday is with the dishwasher.

Are the dishes clean? - No - Run dishwashing loop. Are dishes clean? - Yes - Put dishes away.

This leads in to Loops, and again we can use the dishes analogy.

We may have multiple loads of dishes to put in the dishwasher, so we can call a loop to run (pack the dishwasher and then unload the dishwasher) until there are no dirty dishes left.

Onto the DOM (Document Object Model):

The DOM allows programming languages to interact with the page. The DOM, itself, is not a programming language. But without it languages such as Javascript would not be able to interact with the webpage.
